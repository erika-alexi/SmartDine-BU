# Attributions

The SmartDine BU UI prototype was made in Figma and used as the reference for the interface direction, layout, and feature flow.

Some interface components are based on [shadcn/ui](https://ui.shadcn.com/), which is under the [MIT license](https://github.com/shadcn-ui/ui/blob/main/LICENSE.md). The local implementation has been adapted for the SmartDine BU React/Vite structure and responsive desktop/mobile layouts.

Some food and interface photos used in the prototype and menu seed data are from [Unsplash](https://unsplash.com) under the [Unsplash license](https://unsplash.com/license).

Iconography is provided through [Lucide React](https://lucide.dev/), which is licensed under the ISC license.

The app also uses open-source React ecosystem packages listed in `package.json`, including Vite, Tailwind CSS, Supabase JS, Radix UI primitives, Vitest, and React Testing Library.
